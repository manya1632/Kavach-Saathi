# Storefront reference

The user selected [AshokPrjapati/Meesho-clone](https://github.com/AshokPrjapati/Meesho-clone)
as the base reference for the commerce flow. That repository was inspected at commit
`4057ed337d0d7999d07f7e261b12887c9d904ddb` (Ashok Kumar, 2023).

The referenced repository has no root license file. This storefront therefore uses
its public screen list and general commerce journey as product reference while using
new local source code, mock assets, and API integration written for Kavach Saathi.
No original remote backend credentials or copied source assets are included.
